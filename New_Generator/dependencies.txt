NON RUNNABLE

    draggable_rectangle
        Περιέχει την κλάση που μας επιτρέπει να μετακινούμε και να διαγράφουμε ορθογώνια.

    rectangles_csv_handling
        Περιέχει μία συνάρτηση για την αποθήκευση των ζωγραφισμένων ορθογωνίων
        και μία συνάρτηση για την επανα-ζωγράφιση αποθηκευμένων ορθογωνίων.

    draw_rectangles -> draggable_rectangle, rectangles_csv_handling
        Περιέχει συναρτήσεις για να ζωγραφίζουμε νέα ορθογώνια, να τα μετακινούμε, 
        να τα διαγράφουμε και να τα αποθηκεύουμε. Επιπλέον, μπορεί να επανα-ζωγραφίσει 
        αποθηκευμένα ορθογώνια για να τα επεξεργαστούμε ξανά.

RUNNABLE

    interface_rects -> draw_rectangles
        1) Μας επιτρέπει να δημιουργούμε μία νέα κατανομή αποθηκεύοντας και μια φωτογραφία των ορθογωνίων.
        2) Μας επιτρέπει να επεξεργαζόμαστε μία υπάρχουσα κατανομή ανανεώνοντας και την φωτογραφία των ορθογωνίων.
        3) Μας επιτρέπει να διαγράφουμε πολλές κατανομές ταυτόχρονα, καθώς και τις φωτογραφίες τους.
    
    interface_sets
        1) Μας επιτρέπει να δημιουργούμε ένα dataset για κάθε κατηγορία πλήθους σημείων για τις κατανομές που επιλέγουμε.
        2) Μας επιτρέπει να διαγράφουμε κάθε dataset από κάθε κατηγορία πλήθους σημείων για τις κατανομές που επιλέγουμε.

    interface_points
        Μας επιτρέπει να διαλέγουμε ένα dataset και να το κάνουμε plot.
